iocage.DevfsRules module¶

iocage module for host devfs rule configuration.

class iocage.DevfsRules.DevfsRules(rules_file='/etc/devfs.rules', logger=None)[source]¶

Bases: list

Abstraction for the hosts /etc/devfs.rules.

Read and edit devfs rules in a programmatic way. Restart the devfs service after applying changes.
